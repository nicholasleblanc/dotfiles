# Nicholas' Dotfiles

To update, cd into your local dotfiles repository and then:

```
source install.sh
```

# Install misc items - fonts, themes, etc

```
./misc.sh

```

# Install Homebrew formulae

```
./brew.sh
```

# New system setup

Run scripts in the following order:

1. `source install.sh`
1. `sh init.sh`
1. `sh macos.sh`
1. `sh brew.sh`
1. `sh misc.sh`
